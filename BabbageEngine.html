Analytical Engine on Twitter

Tweet your 'punched cards' to Babbage's Analytical Engine @BabbageEngine and it will return the results. The Babbage engine was created to celebrate Ada Lovelace's Bicentenary on 13th October 2015, and will feature in the Ada Lovelace Event being held at the University of the West of England, Bristol. This software is based on the fourmilab Analytical Engine Simulation. A message must start with @BabbageEngine followed by your line-separated punched cards.

Number Cards

Ncolumn signed-integer
Initialise engine column with a numerical value.
e.g. N1 333

Operation Cards

These control arithmetic operations in the Mill:
+
Add
-
Subtract
*
Multiply
/
Divide

Variable Cards

Transfer columns to the Mill and back:
Lcolumn
Load column into the Mill
Zcolumn
Load and zero column
Scolumn
Save Mill to column

Stepping Cards

<digits
Shift the value in the mill upwards so many digits.
>digits
Shift the value in the mill downwards so many digits.

Combinatorial Cards

Branches may be conditional upon the setting of the Mill's run-up lever.
CB?cards
Conditional Backward branch by number of cards
CF?cards
Conditional Forward branch by number of cards
CB+cards
Unconditional Backward branch by number of cards
CF+cards
Unconditional Forward branch by number of cards

Action Cards

H
Halt and leave card visible
P
Print the result of the last operation by the Mill.
